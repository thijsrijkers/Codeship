<h2>Developing</h2>
Here's how to install the dependencies and run the app locally:

<h3>Vue</h3>
<pre>
npm install -g @vue/cli
# OR
yarn global add @vue/cli</pre>

<pre>
vue --version</pre>

<pre>
npm update -g @vue/cli
# OR
yarn global upgrade --latest @vue/cli</pre>

<h3> Run App</h3>
<pre>
npm run electron:serve</pre>
